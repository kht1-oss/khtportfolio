# 맛킷리스트 디자인 단장 (B: 미니멀 화이트 피드 + 하트) — 설계 문서

작성일: 2026-06-22

## 한 줄 요약
올림푸스 무드를 유지하면서, 배경·카드를 인스타그램처럼 깔끔하게 단장하고
카드 호버 효과 + 하트 추천 + 1위 강조를 추가한다. 기능/데이터/Firebase는 변경 없음.

## 목적 & 범위
- 목적: 단조로운 배경과 카드를 시각적으로 더 매력적으로(인스타 감성) 개선
- 범위 밖(YAGNI): 로그인, 기능 추가, 데이터 모델/Firebase/보안규칙 변경, 레이아웃 구조 변경
- 핵심 원칙: **모양만 바꾸고 동작은 그대로**

## 디자인 상세

### 1. 배경 & 전체 톤
- 배경: 단순 하늘 그라데이션 → **거의 흰색(`#f7f9fc`)의 깔끔한 피드 톤**
- 기존 팔레트 토큰 유지: 강조 남색 `#1f5fa6`, 금빛 `#f0c14b`
- 헤더: 제목 아래 **얇은 금빛 라인**(약 1.5px) 포인트. 신전 기둥 장식은 은은하게 유지하거나 절제
- 레이아웃 구조(시상대 1위 중앙 / 2·3위 양옆 / 4위↓ 격자)와 모바일 1열 전환은 **그대로 유지**

### 2. 카드 스타일 + 호버
- 카드: 흰 배경, 모서리 둥글게(약 12px), 아주 얇은 테두리(`#eef2f7`) + 부드러운 그림자
- 사진: 카드 상단을 꽉 채움(`object-fit: cover`), 카드가 사진 중심이 되도록 텍스트는 절제
- **호버 효과(PC 전용):** `@media (hover: hover)` 안에서만 — 마우스 오버 시 그림자 진해지고 **사진 약 1.04배 확대**(이미지 컨테이너 `overflow:hidden`, `img`에 `transform: scale`). 모바일 터치에는 적용 안 함
- 부드러운 전환: `transition`으로 그림자·확대가 자연스럽게

### 3. 1위 강조
- 1위 카드: 사진 더 크게(기존 시상대 비율 유지) + **👑 왕관** + **은은한 금빛 테두리/그림자**
- 미니멀 톤 유지(과한 효과 지양)

### 4. 하트 추천 버튼
- 표시: **숫자와 함께 하트 아이콘**
  - 아직 추천 안 함(이 기기): **🤍 빈 하트** + 숫자 → 클릭 가능
  - 이미 추천함(이 기기, `hasVoted`): **❤️ 채운 하트** + 비활성(1회 제한 유지)
- 클릭 시: 하트가 🤍→❤️로 바뀌고 숫자 +1, **살짝 톡 튀는 작은 애니메이션**(scale 펄스)
- 동작 로직(`increment(1)`, localStorage 기록, 1회 제한)은 **기존 그대로**, 표시만 하트로 교체

## 동작/데이터 (변경 없음, 확인용)
- 글 작성·추천·삭제·실시간 랭킹·Firestore 저장·보안 규칙: **모두 현행 유지**
- 추천 1회 제한(localStorage), 관리자 암호 삭제: 변경 없음

## 반응형
- 기존 모바일 분기(`@media max-width:560px` 시상대 1열) 유지·점검
- 호버 효과는 PC에서만(`@media (hover: hover)`), 모바일 탭 동작에 영향 없음

## 테스트 (수동 확인 항목)
- 배경/카드가 새 스타일로 보이는지(PC·모바일)
- PC에서 카드 호버 시 그림자·사진 확대가 동작하는지
- 모바일에서 호버 효과 없이 정상으로 보이는지(레이아웃 안 깨짐)
- 추천 안 한 글은 🤍, 누르면 ❤️ + 숫자 +1 + 펄스, 재추천 차단
- 1위 카드에 👑 + 금빛 강조가 보이는지
- 기존 기능(작성/삭제/실시간 갱신)이 그대로 동작하는지
